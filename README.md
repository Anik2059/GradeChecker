## About The Project

GradeChecker demonstrates a straightforward Python Flask app that accepts numeric grades, processes them using backend logic, and displays the appropriate letter grade. It features user input validation and clear error feedback, making it ideal for learning Flask basics and web form handling.

## Getting Started

### Prerequisites

- Python 3.x installed on your system  
- pip for installing packages  

### Installation

```bash
git clone https://github.com/yourusername/gradechecker.git
cd gradechecker
pip install -r requirements.txt
Running the App Locally
bash
Copy code
python app.py
Visit http://localhost:5000 in your web browser to use GradeChecker.

Usage
Enter a mark from 0 to 100 in the input box.

Click the Check Grade button.

The result displays your grade instantly below the form.

Invalid input triggers an error message.
